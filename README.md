# 京东家群 POP 商家打法解析 H5

面向京 ME 客户端网页应用的静态 H5 页面，使用原生 HTML、CSS、JavaScript 和 Vite 构建。

## 本地开发

```bash
npm install
npm run dev
```

## 生产构建

```bash
npm run build
```

构建结果位于 `dist/`。项目已将 Vite `base` 设置为 `./`，静态资源使用相对路径，适合发布在京 ME 网页应用的任意目录下。

## 京 ME 发布

1. 将 `dist/` 部署到可通过 HTTPS 访问的静态托管服务。
2. 在京 ME 开放平台创建或编辑网页应用。
3. 将应用首页配置为部署后的 `index.html` URL。
4. 提交审核并发布。

当前页面不需要登录、分享、通讯录等客户端能力，所以没有引入 JSAPI SDK。后续如增加这些能力，再按京 ME 开放平台文档接入。

## 内容维护

10 个打法的标题和跳转地址集中在 `src/strategies.js`。当前仅“引凤计划”配置了跳转地址，其余入口为空；补充地址后重新执行生产构建即可。
